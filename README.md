# ChatGPT UX Navigator

![Demo](./chatgpt-ux-navigator.png)

A lightweight browser extension that improves navigation within ChatGPT conversations by making it easier to move through chat messages efficiently.

## Features

- Quick navigation between messages
- Improved usability for long conversations
- Minimal footprint, no external dependencies

## Installation

1. Clone the repository.
2. Load it as an unpacked extension in your browser (Chrome/Chromium-based).

## Development

- Built with standard web extension tooling.
- Generated build artifacts are excluded from version control.
- No secrets or environment variables are required.

## Status

Early-stage utility extension focused on UX improvements.
